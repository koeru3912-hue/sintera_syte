# Syntera — ИИ-система записи клиентов

Лендинг для компании Syntera. Один HTML-файл без зависимостей, работает мгновенно.

## Структура проекта

```
syntera-project/
├── index.html          # Основной файл сайта
├── README.md           # Описание проекта
└── materials/
    └── brief.md        # Бриф и исследование ЦА
```

## Технологии
- Чистый HTML + CSS + Vanilla JS
- Без фреймворков и зависимостей
- Шрифт: Manrope (Google Fonts)

## Запуск
Открыть `index.html` в браузере — работает локально без сервера.

## Деплой
Перетащить `index.html` на [app.netlify.com/drop](https://app.netlify.com/drop)

## Сегменты ЦА
- 🦷 Стоматологические клиники
- 🏥 Медицинские центры  
- 💅 Салоны красоты
